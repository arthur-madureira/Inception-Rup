# Inception RUP - Plataforma de Gestão de Editais Públicos

Repositório para a atividade prática da disciplina **Processos de Software** (Unidade 1), focada na **fase de Inception do Rational Unified Process (RUP)**.

## Contexto

Plataforma digital para gerenciar editais públicos de financiamento de projetos esportivos e culturais, apoiando todo o ciclo de vida dos editais e dos projetos financiados.

- **Orçamento máximo**: R$ 300.000,00
- **Prazo máximo**: 8 meses
- **Infraestrutura**: Time e infraestrutura técnica começam do zero
